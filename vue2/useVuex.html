# mapMutations

actions不能直接改变state，只能通过调用mutation来改变state（mutations不能调用mutations，但是actions可以）； mutations只能执行同步操作，而actions可以执行异步操作。

    import { mapMutations } from "vuex";
    <template>
      {{ dataNum }}
      <button @click="addNum">add num</button>
    </template>
store:
    const state = {
    num: 1,
    };
    const mutations = {
     addNum(state) {
        state.num++;
     },
    };

# mapActions

    传递参数直接传 无需做处理会自动映射
    import { mapActions } from "vuex";
      <template>
      <button @click="changeStoreName('传递的参数')">cahnge name</button>
    </template>
    methods: {
      ...mapActions({
        changeStoreName: "changeName",
      }),
   },
store:
   changeName({commit},payload) {
    commit('changeTestName',payload)
   }
