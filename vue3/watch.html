# watch

基础使用

    <script setup lang="ts">
    import { watch, ref } from "vue";
    const fullName = ref(300);
    const res = ref("");
    function sleep(time: any) {
    var timeStamp = new Date().getTime();
    var endTime = timeStamp + time;
    while (true) {
        if (new Date().getTime() > endTime) {
        return;
        }
    }
    }
    const mockData = (newVal: number) => {
    sleep(newVal);
    return "this is res data";
    };
    // 可以直接监听一个 ref
    watch(fullName, async (newVal, oldVal) => {
    try {
        console.log(newVal, oldVal);
        const result = await mockData(newVal); // this is mock settimeout
        res.value = result;
    } catch (error) {
        res.value = "Error!";
    }
    });
    const handleChangeValue = () => {
    fullName.value = 3000;
    };
    </script>

    <template>
    <div>
        {{ fullName }}

        <h1>this is res data: {{ res }}</h1>
        <button @click="handleChangeValue">chang watch value</button>
    </div>
    </template>

# 深层侦听器

    <script setup lang="ts">
        import { watch, reactive } from "vue";
        const obj = reactive({ count: 0 });
        watch(obj, (newValue, oldValue) => {
        console.log(newValue, "newValue");
        console.log(oldValue, "oldValue");
        console.log(newValue == oldValue);
        // 在嵌套的属性变更时触发,任意一个属性发声变化都会触发watch执行
        // 注意：`newValue` 此处和 `oldValue` 是相等的
        // 因为它们是同一个对象！
        });
    </script>

    <template>
        <div>
            {{ obj.count }}
            <button @click="obj.count++">change value</button>
        </div>
    </template>

返回响应式对象的 getter 函数，只有在返回不同的对象时，才会触发回调

    <script setup lang="ts">
        import { watch, reactive } from "vue";
        const state = reactive({
        name: "dawnmale",
        obj: {
            age: 12,
        },
        });
        watch(
            () => state.obj,
            (newValue, oldValue) => {
                // 仅当 state.someObject 被替换时触发
                console.log(newValue, oldValue);
                console.log(newValue == oldValue); //false
            }
        );
            const changeObj = () => {
            state.obj = {
                age: 444,
            };
        };
    </script>

    <template>
    <div>
        {{ state.obj.age }}
        <button @click="changeObj">change value</button>
    </div>
    </template>

深度侦听需要遍历被侦听对象中的所有嵌套的属性，当用于大型数据结构时，开销很大。因此请只在必要时才使用它，并且要留意性能。

# 立即执行

watch 默认是懒执行的：仅当数据源变化时，才会执行回调。但在某些场景中，我们希望在创建侦听器时，立即执行一遍回调。

    watch(source, (newValue, oldValue) => {
    // 立即执行，且当 `source` 改变时再次执行
    }, { immediate: true })

# 触发时机

watch回调默认是在vue更新之前被调用，如果想访问vue更新后的dom

    watch(source, callback, {
        flush: 'post'
    })

    watchEffect(callback, {
         flush: 'post'
    })

    hooks

    import { watchPostEffect } from 'vue'
    watchPostEffect(() => {
        /* 在 Vue 更新后执行 */
    })
